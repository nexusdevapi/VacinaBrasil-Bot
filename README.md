# Objetivo do Projeto

Desenvolver um assistente virtual que utilize dados de portais públicos oficiais de saúde sobre vacinação para informar o cidadão sobre:

* Calendário vacinal para diferentes faixas etárias (crianças, adultos, idosos, etc.);
* Cobertura vacinal em diferentes regiões do Brasil;
* Demais informações relevantes disponíveis sobre o tema.

O sistema deve:

* Integrar-se com plataformas amplamente utilizadas, como o Telegram;
* Utilizar linguagem natural para interação com o usuário;
* Oferecer suporte a comandos e, como diferencial, interação por voz (semelhante a assistentes virtuais como a Alexa).

Ressalta-se que todas as funcionalidades devem ser desenvolvidas por meio de programação própria, sem a utilização de APIs externas.
